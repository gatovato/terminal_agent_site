Terminal Agent began as a project to learn programming. It developed into a passionate love for Linux and a desire to give back. I hope that Terminal Agent can help anyone interested in learning the concepts of command line operation and Linux commands and hopefully develop the same love for Linux as myself.


Background and Features


With the latest virtual reality technology an up and coming Agent at the best security Agency has been assigned to clear out a severely infected server rumored to be compromised by the radical hacker Zardo.
